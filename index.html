Tentang Event

Apa itu GDGoC Web Summit?

GDGoC Web Summit adalah konferensi tahunan yang diselenggarakan oleh Google Developer Groups on Campus Jember. Event ini dirancang khusus untuk mahasiswa dan pemula yang ingin terjun ke dunia web development secara serius.
